🌞 Hackathon Solar Insight Dashboard

This project delivers real-time solar energy condition predictions, live weather monitoring, and interactive insights using machine learning models trained on collected environmental data.

Key Features 🚀

  • 📡 Live weather data collection and visualization
  • 🤖 Real-time machine learning predictions for solar energy conditions
  • 🗺️ Interactive maps for solar radiation, energy production, temperature, and cloud cover
  • 📈 Dynamic chart tracking solar conditions over time

How It Works 🛠️

  1. 🌦️ Collect live weather data every minute
  2. 💾 Save live data into a CSV database
  3. 🤖 Train a machine learning model (Random Forest Classifier)
  4. ⚡ Predict solar conditions from real-time inputs
  5. 📈 Update dashboard and charts live with insights

Technologies Used 🔧

Python • FastAPI • Machine Learning • Chart.js • OpenWeatherMap API • Open-Meteo API • Leaflet.js
